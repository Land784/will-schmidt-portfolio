# William Schmidt Portfolio

A modern personal portfolio website built with Next.js, React, TypeScript, and Tailwind CSS.

This website serves as a central hub for my projects, experience, resume, and technical interests as I continue developing software engineering, AI, and systems programming skills.

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel

---

## Features

- Modern responsive UI
- Custom hero section
- About Me page
- Experience timeline
- Embedded downloadable resume
- Reusable React component structure
- Fully deployed custom domain

---

This project was created to:

## Local Development

Clone the repository:

```bash
git clone https://github.com/Land784/will-schmidt-portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the application in your browser:

```text
http://localhost:3000
```

---

## Deployment

The site is deployed using Vercel and connected to a custom domain.
